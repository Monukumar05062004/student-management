# 📘 Student Management System (Console-Based)

## 📌 Description

The **Student Management System** is a console-based Java application designed to perform basic CRUD (Create, Read, Update, Delete) operations on student records.

This project demonstrates a strong understanding of **core Java concepts**, **data structures**, and **layered architecture**, making it suitable for beginners transitioning into backend development.

---

## 🎯 Key Features

* ➕ Add a new student
* 📋 View all students
* 🔍 Search student by ID
* ✏️ Update student details
* ❌ Delete student record
* ⚠️ Input validation and error handling
* 📦 Structured response handling using custom Response class

---

## 🛠️ Technologies Used

* Java (Core Java)
* OOP (Object-Oriented Programming)
* Collection Framework (HashMap / LinkedHashMap)

---

## 🏗️ Project Architecture

The project follows a **layered architecture** similar to real backend systems:

* **Controller Layer** → Handles user input/output
* **Service Layer** → Contains business logic
* **Repository Layer** → Manages data storage
* **Model Layer** → Represents Student entity

---

## 🧠 Concepts Demonstrated

* Encapsulation
* Separation of Concerns
* Exception Handling
* Data Management using Collections
* Clean Code Practices
* Custom Response Design (success & failure handling)

---

## 🚀 How to Run the Project

1. Clone the repository
2. Open in any Java IDE (IntelliJ / Eclipse)
3. Run the main class (`StudentController`)
4. Follow the menu-driven options in the console
